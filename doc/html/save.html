NAME

      save - a module to access persistent memory

SYNOPSIS

      local SAVE = dofile('./AI/USER_AI/lib/save.lua')

DESCRIPTION

    save is a module that implements a simple API to access memory in disk space. The data is saved as a lua file at "./AI/USER_AI/save/OWNER_GID-HOMUN_CLASS/" directory.

    If the homunculus evolves and the previous data is required, you must rename the with its new homunculus class.

FUNCTIONS

get(actor, key)

Reads the data pointed by key.

actor should be the homunculus/mercenary actor object.

put(actor, key, data)

Writes data in a lua file and indexes it by key.

actor should be the homunculus/mercenary actor object.
