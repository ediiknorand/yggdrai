NAME

     getActors, newGroup - create an actor group

SYNOPSIS

     local actors = getActors([foreach])
     local actors = newGroup()

DESCRIPTION

    The getActors() function returns an actor group table that contains all the detectable actors. Optionally, it can receive a foreach() function that executes for each actor.

    The newGroup() function returns an empty actor group table.

Attributes

target

A group formed from all targets of the original group.

n

Size of the group.

Methods

nearest(actor)

Returns the nearest actor from actor in the actor group.

Operators

Addition (+)

If the right operand is an actor object, returns a copy of the actor group with this actor object inserted.

If the right operand is an actor group, returns the union of the two groups.

Subtraction (-)

If the right operand is an actor object, returns a copy of the actor group without the actor object.

If the right operand is an actor group, returns a copy of the first group without the actors from the second group.

Multiplication (*)

Returns the intersection between two actor groups.

Division (/)

Returns a new actor group filtered from the original actor group by their types. The right operand (the filter) can be a string ('player', 'summon', 'pet', 'npc', 'warp', 'monster' or 'unknown'), a number (their class id), an actor group or a table that contains actor objects or strings.

Exponentiation (^)

Returns a new actor group filtered from the original actor group filtered by the type of their targets.

The right operand (the filter) can be a number (their class id, or -1 to represent nothing), a string ('player', 'summon', 'pet', 'npc', 'warp', 'monster' or 'unknown'), an actor object, an actor group or a table that contains some of the previously mentioned filters.

Unary minus (-)

Returns a new actor group that contains all detectable actors but those who are in the actor group.
