NAME

     merclass - a module that tries to fix the Mercenary AI behavior

SYNOPSIS

     local MERCLASS = dofile('./AI/USER_AI/lib/merclass.lua')

DESCRIPTION

    merclass is a module that implements an actor object with a different class method. The original class only works with homunculus (and the correspondent GetV call) or can only recognize only the Mercenary's class.

    Important note: Currently, only suports players.

FUNCTIONS

getActor(gid)

Creates an actor object from the gid gid.

This actor object has a special class method, that tries to guess the actor's class (since their original GetV call only works with homunculus).

getActors([foreach])

Creates an actor group object from the all detectable actors with the custom actor class method. Optionally, it can receive a foreach function that executes for each actor.

shift(n)

Sets the difference between GID_BASE (default: 110000000) and the gid of the first spawned mob (default: Prontera's Wild Rose).

ROversion(version)

Sets the major RO update version for compatibility.

version should be 'pre-renewal' or 'renewal.'
