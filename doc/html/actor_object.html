NAME

     getActor, newActor - create an actor object

SYNOPSIS

     local actor = getActor(gid)
     local actor = newActor()

DESCRIPTION

    The getActor() function returns a new actor object with the gid gid.

    The newActor() function returns an empty actor object which represents no actual actor.

Attributes

pos

An array that contains the actor's x,y coordinates.

xy

Same as pos.

x

Actor's x coordinate.

y

Actor's y coordinate.

target

Actor's target (also an actor object).

motion

A number that represents the actor's current motion value. (See const)

dead

A boolean value that states the current health condition of actor. (or if their are missing)

owner

Actor's owner. (Note: It'll only return the player's gid)

class

A number that represents the characters class. (See const)

mobid

Same as class.

rider

A boolean value that represents if actor is riding (or if it's a flying taekwon master).

range

Actor's attack range.

hp

Actor's current HP.

sp

Actor's current SP.

maxhp

Actor's maximum HP.

maxsp

Actor's maximum SP.

merclass

Actor's class. (See const)

canAttack

A boolean value that states if the actor is attackable. (More precisely, if the player's cursor becomes a sword icon)

t

Time in seconds since the first time this actor was detected.

Methods

dist(actor)

Returns the distance between the actor whose the method was called and actor.

skillrange(skill, skill_level)

Returns the maximum distance between the homunculus/mercenary and the actor necessary to cast skill with level skill_level.

sr(skill, skill_level)

Same as skillrange().

skillrangepos(skill, skill_level)

Returns an array that contains what'll be the new position of the homunculus/mercenary if they cast skill with level skill_level at the actor. (Note: Only available in renewal)

srxy(skill, skill_level)

Same as skillrangepos().

srx(skill, skill_level)

Returns the x coordinate of the new position of the homunculus/mercenary if they cast skill with level skill_level at the actor.

sry(skill, skill_level)

Returns the y coordinate of the new position of the homunculus/mercenary if they cast skill with level skill_level at the actor.

Operators

Addition (+)

If the right operand is an actor object, returns a new actor group with both actors in it.

Equals (==)

Compares if both actor objects correspond to the same actor.

Exponentiation (^)

Return a boolean value that states if the actor has the second operand as a target.

If the second operand is a string ('player', 'summon', 'pet', 'monster', 'npc', 'warp' or 'unknown'), tests the target's type. If the second operand is an number, tests the target's class. If the second operand is an actor object, tests if this actor object is the actor's target.
