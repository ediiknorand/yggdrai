NAME

     state - an useful module with set of state functions

SYNOPSIS

     local ST = dofile('./AI/USER_AI/lib/state.lua')

DESCRIPTION

    state is a build-in module that implements useful state functions to be used inside profile files.

STATES

hold(actor)

Moves actor to its own position.

move(actor, x, y)

Moves actor to the cell specified by the x,y coordinates.

attack(actor, target)

Forces actor to attack target, if available.

follow(actor)

Follows actor's owner.

chase(actor, target)

Forces actor to follow/chase target.

skill(actor, skill_level, skill, target)

Forces actor to cast a skill, specified by skill (its id), at the level skill_level at the actor object target. If target was generated by newActor() function, actor casts a ground skill specified by target coordinates.

patrol(actor)

Moves actor into a circle around its owner.

METHODS

copy(state)

Creates a copy of state that can be used for different transitions.
