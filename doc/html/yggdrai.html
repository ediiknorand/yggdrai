NAME

     yggdrai = initialize and execute a YggdrAI processing tick

SYNOPSIS

     local yggdrai = dofile('./AI/USER_AI/yggdrai.lua')

     yggdrai(path_to_profile, my_gid)

DESCRIPTION

    The yggdrai() function initializes, during its first call, and executes a YggdrAI processing tick.

    Loading yggdrai.lua also declares functions such as getActor, newActor, getActors, newGroup to Lua's global envionment. require() is also changed upon yggdrai.lua loading to a more version-friendly function.

    During its first call, yggdrai() loads a profile file specified by path_to_profile and generates an actor object from my_gid. This actor object is passed as argument to the functions defined by profile.

    yggdrai() also manages the state transitions and the command handlers defined by profile. Each processing tick executes the current state transition function and the current stage function.
